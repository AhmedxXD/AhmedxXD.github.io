Electronics Project

1- Card based Attendance System

The main components of this project are the ESP32 controller and RFID module. This project stores student IDs in a Google Sheet for specific days, making attendance tracking fully automated and easy to manage.

2- Sequence Game

The main components of this project are the ESP32 controller, joystick module, and LEDs. This project challenges players to follow a sequence of lights and movements, improving memory and reaction skills in an interactive and fun way.
